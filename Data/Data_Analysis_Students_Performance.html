Common Pandas Functions Explained

  • df.head() Shows the first 5 rows of the dataset to get a quick preview of the data.
  • df.tail() Displays the last 5 rows of the dataset, useful for checking the end of the data.
  • df.info() Summarizes the dataset, showing column names, data types, and missing values.
  • df.shape Returns the number of rows and columns in the dataset as a tuple.
  • df.columns Lists all the column names in the dataset.
  • df.dtypes Shows the data type (e.g., int, float, object) for each column.
  • df.describe() Provides summary statistics (like mean, min, max) for numerical columns.
  • df.isnull() Returns a DataFrame showing where missing (NaN) values are located (True/False).
  • df.rename() Renames columns or indexes using a dictionary of old and new names.
  • df.sort_values() Sorts the DataFrame by one or more columns, either ascending or descending.
  • df.loc[] Accesses rows or columns by label (name).
  • df.iloc[] Accesses rows or columns by index position.
  • df['column'].unique() Returns all unique values found in a specific column.
  • df.sample() Randomly selects rows from the dataset, useful for quick previews or testing.

Missing Values and How to Check for Them

Missing values occur when some data is not recorded or is unavailable, and they are often represented as NaN (Not a Number) in a dataset. These missing entries can affect the accuracy of your analysis or any machine learning models you might build, so it's important to detect and handle them appropriately.

In pandas, you can check for missing values using the df.isnull() function, which returns a DataFrame showing True for missing values and False otherwise. To get a quick summary of how many values are missing in each column, you can use df.isnull().sum(), which helps identify which columns might need cleaning or imputation before analysis.

Well calculate average scores based on test preparation course.

What is Group Analysis and groupby in Pandas?

Group analysis is a technique used in data analysis to summarize or compare data based on categories (groups). For example, you might want to compare average math scores between males and females, or see how students from different education backgrounds perform.

In pandas, we use the .groupby() function to do this. It allows you to:

  • Group data based on the values in one or more columns.
  • Apply a function like .mean(), .sum(), .count(), etc., to each group.
In[21]:
df.groupby('gender')['math_score'].mean()
Out[21]:
gender
female    63.633205
male      68.728216
Name: math_score, dtype: float64

This line of code:

  • Groups the data by the gender column.
  • Calculates the average (mean) math score for each gender group.

Why is it useful?

  • It helps reveal patterns or differences between groups.

  • Its used in exploratory data analysis to ask and answer questions like:

  • Does test preparation improve scores?

  • Do students from different ethnic groups perform differently?

  • Does gender affect the average score?

Step 9: Create New Features

In data analysis, creating new features means generating additional columns in your dataset that are derived from existing data. This helps uncover hidden patterns and makes the dataset more useful for analysis or machine learning.

Why create new features?

  • To summarize multiple columns (e.g., average of test scores)
  • To categorize values (e.g., pass/fail based on score)
  • To add context or make comparisons easier
In[22]:
df['average_score'] = df[['math_score', 'reading_score', 'writing_score']].mean(axis=1)
